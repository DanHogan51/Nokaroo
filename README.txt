
# Nokaroo — Directory Starter (No-Ads Launch)

You asked to launch **content-first** with no advertising. This bundle removes the "Advertise With Us" section and navigation. Everything else is identical and ready for Vercel.

## Deploy to Vercel

1. Make a new GitHub repo and upload these files (root of repo).
2. In Vercel → New Project → Import Repo.
3. Framework: **Other**, Build: **None**, Output: **/**.
4. Deploy → `https://<project>.vercel.app`

## Editing

- Listings live in `listings.json`.
- SEO files: `robots.txt` and `sitemap.xml`.
- GA4 placeholder is in `index.html` (optional).
